title=Trait Tag Framework
description=A Project Zomboid framework for adding user-defined strings correlating to player traits. Trait tags are stored in a table accessed by a module and has built-in functions.
description=

description=[h1]How to Use:[/h1]
description=Declare the module before it's needed (typically at the beginning of the file) with
description=[code]local TTF = require("TraitTagFramework");[/code]
description=Depending on your use case, you'll want to add your tags before the player is created. To do this, make a function that gets added to the OnGameBoot event with
description=[code]Events.OnGameBoot.Add([your function here])[/code]
description=If you're creating new traits, feel free to mingle  your trait and tag declarations together, like so:
description=[code]local AT_Carpenter = TraitFactory.addTrait("AT_Carpenter", getText("UI_trait_AT_Carpenter"), 2, getText("UI_trait_AT_Carpenter_desc"), false);
description=AT_Carpenter:addXPBoost(Perks.Woodwork, 1)
description=[b]TTF.Add("AT_Carpenter", "Anthro,HabitatBuilder");[/b][/code]
description=
description=[h1]Available Functions[/h1]
description=[list]
description=[*][h3]ToString()[/h3]
description=Returns the entirety of the table and tags in a string.
description=[*][h3]TagTableToString(string traitName)[/h3]
description=Returns the tags associated with a trait as a string
description=[*][h3]Add(string traitName, string tags)[/h3]
description=Add(s) tag(s) to the TraitTags module.
description=[b]Note: For tags, use alphanumeric symbols only, and separate each tag with a comma.[/b]
description=[*][h3]Remove(string traitName)[/h3]
description=Removes a trait and all of its tags from the TraitTags module.
description=[*][h3]RemoveTag(string traitName, string tag)[/h3]
description=Removes the specified tag from the specified trait in the TraitTags module.
description=[*][h3]GetTable()[/h3]
description=Returns the entirety of the TraitTags table as a table.
description=[*][h3]GetTagTable(string traitName)[/h3]
description=Returns the table of tags associated with a trait as a table.
description=[*][h3]GetTagStatistics()[/h3]
description=Returns a string containing each tag used, and how many times they're used.
description=Note: I find this helpful for balancing purposes.
description=[*][h3]GetPlayerTraitTags(Player player)[/h3]
description=Returns a string containing each tag a player has.
description=[*][h3]GetPlayerTagStatistics(Player player)[/h3]
description=Returns a string containing each tag and the amount(count) of each the player has. This list in the string is ordered by count.
description=String format: "[tag]: [count]; [tag]: [count];..."
description=[*][h3]PlayerHasTag(Player player, string targetTag)[/h3]
description=Returns a boolean, which is true if the player has the targetTag.
description=[*][h3]PlayerTagCountLargerThan(Player player, string subjectTag, string comparatorTag)[/h3]
description=Returns a boolean, which is true only if the subjectTag's count is larger than the comparatorTag's count.
description=Note: This will also return false if one of the tags cannot be found (with a console print warning of a nil if debug mode is on).
description=[*][h3]GetAllTraitsWithTag(string subjectTag)[/h3]
description=Returns a collection of Traits that have the subject tag.
description=[/list]
